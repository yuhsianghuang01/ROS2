🚀 第一部分：快速上手指南

💡 什麼是 nano？

nano 是一個簡單易用的終端文字編輯器，特別適合 Linux 初學者使用。它的設計理念是「簡單直觀」，所有常用功能的快捷鍵都會顯示在螢幕底部。

🎯 nano 的優點：
  • 學習曲線平緩，容易上手
  • 快捷鍵顯示在畫面底部，不用背誦
  • 輕量級，啟動快速
  • 大部分 Linux 發行版預設安裝
  • 支援語法高亮和基本編輯功能

📦 安裝 nano

大部分 Linux 發行版都預設安裝了 nano，如果沒有，可以使用以下命令安裝：

# Ubuntu/Debian
sudo apt install nano

# CentOS/RHEL/Fedora
sudo yum install nano
# 或
sudo dnf install nano

# Arch Linux
sudo pacman -S nano

檢查是否已安裝：

nano --version

🔧 基本使用方法

啟動 nano

# 建立新檔案
nano

# 開啟現有檔案
nano filename.txt

# 以唯讀模式開啟
nano -v filename.txt

# 顯示行號
nano -l filename.txt

# 開啟檔案並跳到第10行
nano +10 filename.txt

nano 介面說明

  • 頂部：顯示 nano 版本和檔案名稱
  • 中間區域：文字編輯區
  • 底部兩行：顯示常用快捷鍵
    • ^ 代表 Ctrl 鍵
    • M- 代表 Alt 鍵（在某些鍵盤配置中可能是 Meta 鍵，例如 macOS 或特定的 Linux 系統）。Alt 鍵常用於執行進階功能，例如在 nano 中使用 Alt + W 繼續搜尋或 Alt + R 啟用正規表示式模式。

⌨️ 必學快捷鍵

快捷鍵 功能 說明
Ctrl + O 儲存檔案 Write Out，會提示輸入檔案名稱
Ctrl + X 退出 nano 如有未儲存變更會提示儲存
Ctrl + W 搜尋文字 Where Is，輸入要搜尋的文字
Alt + R 啟用正規表示式 在搜尋模式下啟用 regex 語法（注意：GNU nano 的正規表示式支援有限，\d 可能無法匹配數字，請使用 [0-9] 替代）
Alt + R 查找 "\w" 啟用正規表示式後，輸入 [a-z] 查找匹配的單字字符，例如字母、數字或底線
\d 數字 匹配任意一個數字（0-9）
\w 單字字符 匹配任意一個字母、數字或底線
\s 空白字符 匹配空格、Tab 或其他空白字符
. 任意字符 匹配除換行符以外的任意單個字符
^ 行首匹配 匹配行的開頭
$ 行尾匹配 匹配行的結尾
[abc] 字符集 匹配 a、b 或 c 中的任意一個字符
[a-z] 字符範圍 匹配小寫字母 a 到 z 中的任意一個字符
(...) 分組 將匹配的內容分組，便於後續引用
\b 單詞邊界 匹配單詞的開頭或結尾
\* 零次或多次 匹配前一個字符零次或多次
\+ 一次或多次 匹配前一個字符一次或多次
\? 零次或一次 匹配前一個字符零次或一次
\| 或 匹配左邊或右邊的表達式
Alt + R 查找 "\d2" 啟用正規表示式後，輸入 \d2 查找匹配的數字後接 2 的內容，例如 "123asdf45624" 中的 "24"
步驟 使用鍵盤操作查找 regex
  1. 按 Ctrl + W 開啟搜尋模式。
  2. 按 Alt + R 啟用正規表示式模式。
  3. 輸入 \d2 作為搜尋條件。
  4. 按 Enter 開始搜尋。
  5. 使用 Alt + W 繼續搜尋下一個匹配項目。
  6. 使用 Alt + B 啟用或停用備份檔案功能（Backup files enabled/disabled）。
  7. 使用 Alt + B 向上查找符合的結果。
Alt + W 繼續搜尋下一個 搜尋下一個相符的結果
Alt + B 搜尋上一個 搜尋上一個相符的結果
Ctrl + K 剪下整行 Cut Text，剪下游標所在行
Ctrl + U 貼上 Uncut Text，貼上剪下的內容
Ctrl + G 顯示說明 Get Help，顯示所有快捷鍵
Alt + U 復原 Undo，復原上一個操作
Alt + E 重做 Redo，重做已復原的操作

📝 第一個編輯範例

🎯 練習：建立一個簡單的文字檔
  1. 開啟終端機
  2. 輸入 nano hello.txt
  3. 輸入一些文字：「Hello, nano!」
  4. 按 Ctrl + O 儲存
  5. 按 Enter 確認檔案名稱
  6. 按 Ctrl + X 退出
  7. 用 cat hello.txt 確認檔案內容

⚙️ 設定與自訂

nano 設定檔

nano 的設定檔位於：

  • 系統全域設定：/etc/nanorc
  • 使用者個人設定：~/.nanorc
# 建立個人設定檔
nano ~/.nanorc

常用設定選項

# 顯示行號
set linenumbers

# 自動縮排
set autoindent

# 設定 Tab 大小為 4 個空格
set tabsize 4

# 將 Tab 轉換為空格
set tabstospaces

# 啟用軟換行
set softwrap

# 啟用滑鼠支援
set mouse

# 顯示標題列
set titlebar

# 顯示狀態列
set statusbar

# 啟用語法高亮
include "/usr/share/nano/*.nanorc"

# 設定備份目錄
set backupdir "/tmp"

# 建立備份檔案
set backup

# 平滑捲動
set smooth

# 在搜尋時不區分大小寫
set casesensitive

語法高亮設定

nano 支援多種程式語言的語法高亮：

# 檢視可用的語法高亮檔案
ls /usr/share/nano/

# 常見的語法檔案
/usr/share/nano/python.nanorc # Python
/usr/share/nano/html.nanorc # HTML
/usr/share/nano/css.nanorc # CSS
/usr/share/nano/javascript.nanorc # JavaScript
/usr/share/nano/c.nanorc # C/C++
/usr/share/nano/sh.nanorc # Shell script

💡 實用技巧

1. 命令列選項

# 開啟檔案並跳到第 20 行
nano +20 filename.txt

# 以唯讀模式開啟
nano -v filename.txt

# 顯示行號
nano -l filename.txt

# 不使用設定檔
nano -I filename.txt

# 不自動換行
nano -w filename.txt

# 建立備份檔案
nano -B filename.txt

# 設定 Tab 大小
nano -T 4 filename.txt

2. 多檔案編輯

📁 多檔案操作：
  • nano file1.txt file2.txt：同時開啟多個檔案
  • Alt + .：切換到下一個檔案
  • Alt + ,：切換到上一個檔案
  • Ctrl + R：插入其他檔案的內容

3. 文字選取技巧

  1. 按 Ctrl + 6 開始選取
  2. 移動游標選取文字範圍
  3. 按 Ctrl + K 剪下選取的文字
  4. 按 Ctrl + U 貼上

4. 搜尋技巧

🔍 進階搜尋：
  • 按 Ctrl + W 開啟搜尋
  • 按 Alt + R 啟用正規表示式
  • 按 Alt + C 區分大小寫
  • 按 Alt + B 反向搜尋

5. 檔案處理技巧

# 檢視檔案內容而不編輯
nano -v filename.txt

# 將輸出導向到 nano
echo "Hello World" | nano

# 編輯管道輸出
ps aux | nano

# 編輯系統設定檔（需要 sudo）
sudo nano /etc/hosts

🔧 故障排除

常見問題與解決方案

❌ 問題：nano 顯示亂碼
✅ 解決：
  • 確認檔案編碼：file filename.txt
  • 轉換編碼：iconv -f big5 -t utf-8 filename.txt
  • 設定環境變數：export LANG=zh_TW.UTF-8
❌ 問題：無法儲存檔案
✅ 解決：
  • 檢查檔案權限：ls -l filename.txt
  • 使用 sudo：sudo nano filename.txt
  • 檢查磁碟空間：df -h
❌ 問題：快捷鍵不起作用
✅ 解決：
  • 檢查終端機設定
  • 確認沒有與其他程式衝突
  • 重設 nano 設定：nano -I

備份與復原

💾 自動備份設定：
# 在 ~/.nanorc 中加入
set backup
set backupdir "/home/username/nano-backups"

這樣 nano 會在每次儲存時建立備份檔案，檔名會加上 ~ 後綴。

何時選擇 nano？

🎯 nano 最適合的場景：
  • SSH 遠端連線：編輯伺服器上的設定檔
  • 快速編輯：簡單的文字檔修改
  • 學習 Linux：初學者友善的編輯器
  • 系統管理：編輯系統設定檔
  • 腳本編寫：簡單的 shell script 編寫
🚀 提升效率的最後建議：
  1. 熟記基本快捷鍵（Ctrl+O, Ctrl+X, Ctrl+W）
  2. 設定個人化的 ~/.nanorc 設定檔
  3. 善用語法高亮功能
  4. 練習搜尋與替換功能
  5. 學會使用多檔案編輯功能
